About Offline Browser

Offline Browser is a SMS Based Text Message Browser that Lets You to Access Internet Without the Need Of Internet Connectivity,The Application Renders the Requested Web Pages By Removing the Multimedia,Web Scripts and Other Surplus HTML Contents to Compress the Page to the Maximum Limit Possible.Thereby Fetching You the Result in Plain Text Within a Moment.

This Application Is Developed for the Final Year Project In Information Science And Engineering By Ashif Ismail,Rohan D'Souza And Mohammed Shafi.P Under the Guidance of Mr_____

How Application Works

Working of Offline Browser is Fairly Simple.Whenever A User Enters the Web Address To the Address Bar In the App,The App Sends this Address As SMS to Our Number,Which Is Configured to Forward this Address As a HTTP POST Request to Our Node JS Backend,The Backend Makes a HTTP GET Request to the Desired Address and then Extracts the Source of the Requested URL and Compress The Source by Truncating All the Multimedia,CSS,Java Script Contents to get the Complete Textual Contents of the Website and Sends this Result As SMS to the Requested User.The User Recieves this Result As A Stream Of SMS's Until the Full Webpage Is Displayed.
